# AI Assistant

ChatGPT стиліндегі студенттік веб сайт.

## Беттер
- index.html
- chat.html
- features.html
- contact.html

## Ішінде бар
- Беттер арасында навигация мәзірі
- Бет ішіндегі навигация
- ChatGPT стиліндегі demo AI чат
- 3 AI сурет
- 1 аудиофайл
- 1 бейнефайл
- 1 кері байланыс формасы
- JavaScript арқылы жұмыс істейтін интерфейс
- GitHub Pagesке дайын статикалық сайт

## Ескерту
Бұл толық нейрондық AI емес. GitHub Pagesта APIсыз жұмыс істейтін демо көмекші.
Толық ChatGPT деңгейіндегі AI үшін сервер және AI API қажет.
