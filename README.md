# Qazaq Nature

Бұл GitHubқа жүктеуге жеңілдетілген нұсқа.

## Қалай жүктеу керек
1. Архивті толық шығарып алыңыз.
2. Шыққан папканы ашыңыз.
3. Ішіндегі барлық файлды Ctrl + A арқылы таңдаңыз.
4. GitHub Upload files бетіне сүйреп апарыңыз.
5. Commit changes басыңыз.

## Маңызды
GitHubқа ZIP архивтің өзін жүктемеңіз. Тек архив ішіндегі файлдарды жүктеңіз.
